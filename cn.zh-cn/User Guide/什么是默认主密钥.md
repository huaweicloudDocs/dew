# 什么是默认主密钥？<a name="dew_01_0045"></a>

默认主密钥，是对象存储服务（Object Storage Service，OBS）等其他云服务自动通过密钥管理为用户创建的用户主密钥，其别名后缀为“/default”。

默认主密钥可通过密钥管理界面进行查询，不支持禁用、计划删除操作。
