# 查询配额<a name="ShowUserQuotas"></a>

## 功能介绍

查询配额，查询用户可以创建的用户主密钥配额总数及当前使用量信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=KMS&api=ShowUserQuotas)中调试该接口。

## URI

GET /v1.0/\{project\_id\}/kms/user-quotas

## 响应示例

**状态码为 200 时: **

请求已成功。

```
{
  "quotas" : {
    "resources" : [ {
      "quota" : 20,
      "used" : 20,
      "type" : "CMK"
    }, {
      "quota" : 100,
      "used" : 0,
      "type" : "grant_per_CMK"
    } ]
  }
}
```

**状态码为 403 时: **

认证失败。

```
{
  "error" : {
    "error_code" : "KMS.XXX",
    "error_msg" : "XXX"
  }
}
```
