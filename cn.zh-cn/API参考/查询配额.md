# 查询配额<a name="dew_02_0025"></a>

## 功能介绍<a name="section27849192112353"></a>

查询配额，查询用户可以创建的用户主密钥配额总数及当前使用量信息。

>![](public_sys-resources/icon-note.gif) **说明：**   
>用户可以创建的用户主密钥配额不包括服务自动创建的默认主密钥数量。

## 示例<a name="section169469372415"></a>

-   请求样例

    无

-   响应样例

    ```
    {
        "quotas": {
            "resources": [
                {
                    "type": "CMK",
                    "used": 15,
                    "quota": 20
                },
                {
                    "type": "grant_per_CMK",
                    "used": 15,
                    "quota": 100
                }
    
            ]
        }
    }
    ```

    或

    ```
    {
        "error": {
            "error_code": "KMS.XXXX",
            "error_msg": "XXX"
        }
    }
    ```
