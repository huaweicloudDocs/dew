# 下载凭据备份<a name="DownloadSecretBlob"></a>

## 功能介绍

下载指定凭据的备份文件

## 接口约束

此接口返回的信息为表示凭据备份文件的字符串，内容加密格式，不可读。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://console.huaweicloud.com/apiexplorer/#/openapi/CSMS/doc?api=DownloadSecretBlob)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI<a name="atuogenerate_2"></a>

POST /v1/\{project\_id\}/secrets/\{secret\_name\}/backup
