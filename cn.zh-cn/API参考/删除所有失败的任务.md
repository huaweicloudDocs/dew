# 删除所有失败的任务<a name="DeleteAllFailedTask"></a>

## 功能介绍

删除操作失败的任务信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=KPS&api=DeleteAllFailedTask)中调试该接口。
