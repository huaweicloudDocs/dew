# 查询指定版本信息<a name="ShowVersion"></a>

## 功能介绍

查指定API版本信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=KMS&api=ShowVersion)中调试该接口。

GET /\{version\_id\}

## 响应示例

**状态码为 200 时: **

查指定API版本信息。

```
{
  "version" : {
    "min_version" : "",
    "links" : [ {
      "rel" : "self",
      "href" : "https://kms.region_id.domain.com/v1.0/"
    } ],
    "id" : "v1.0",
    "version" : "",
    "updated" : "2016-10-29T02:00:00Z",
    "status" : "CURRENT"
  }
}
```
