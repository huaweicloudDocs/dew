# 修改密钥描述<a name="UpdateKeyDescription"></a>

## 功能介绍

-   功能介绍：修改用户主密钥描述信息。

-   说明：

    -   服务默认主密钥（密钥别名后缀为“/default”）不可以修改。

    -   密钥处于“计划删除”状态，密钥描述不可以修改。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://console.huaweicloud.com/apiexplorer/#/openapi/KMS/doc?api=UpdateKeyDescription)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。
