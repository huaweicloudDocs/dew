# 恢复凭据对象<a name="UploadSecretBlob"></a>

## 功能介绍

通过上传凭据备份文件，恢复凭据对象

## 接口约束

此接口返回的信息为凭据的元数据信息，不包含凭据值。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://console.huaweicloud.com/apiexplorer/#/openapi/CSMS/doc?api=UploadSecretBlob)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI<a name="atuogenerate_2"></a>

POST /v1/\{project\_id\}/secrets/restore
