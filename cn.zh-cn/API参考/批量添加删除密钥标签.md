## 示例<a name="section207874128353"></a>

如下以添加标签键为“key1“和“key“，标签值为“value1“和“value3“为例。

-   请求样例

    ```
    { 
        "action": "create",
        "tags": [
            {
                "key": "key1",
                "value": "value1"
            },
            {
                "key": "key",
                "value": "value3"
            }
        ]
    }
    ```

    或

    ```
    {
        "action": "delete",
        "tags": [
            {
                "key": "key1",
                "value": "value1"
            },
            {
                "key": "key2",
                "value": "value3"
            }
        ]
    }
    ```


-   响应样例

    ```
    { 
    }
    ```

    或

    ```
    {    
           "error": {        
                  "error_code": "KMS.XXXX",        
                  "error_msg": "XXX"     
         } 
    }
    ```
