## 示例<a name="section12430172338"></a>

如下以查询项目标签为例。

-   请求样例

    无

-   响应样例

    ```
    {            "tags": [
                 { 
                   "key": "key1",
                   "values": [
                       "value1",
                       "value2"
                     ]
                  },
                 {
                   "key": "key2",
                   "values": [
                       "value1",
                       "value2"
                    ]
            }
    ]
    }
    ```

    或

    ```
    {    
           "error": {        
                  "error_code": "KMS.XXXX",        
                  "error_msg": "XXX"     
         } 
    }
    ```
