# 查询任务信息<a name="ListKeypairTask"></a>

## 功能介绍

根据SSH密钥对接口返回的task\_id，查询SSH密钥对当前任务的执行状态。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://console.huaweicloud.com/apiexplorer/#/openapi/KPS/doc?api=ListKeypairTask)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。
