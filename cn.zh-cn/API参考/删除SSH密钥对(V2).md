# 删除SSH密钥对<a name="dew_02_0204"></a>

## 功能介绍<a name="s9636d7e16b064e68a5c68d5c50a738e2"></a>

根据SSH密钥对的名称，删除指定SSH密钥对。

## URI<a name="sce81dd29969b483b929fba7c1713dacf"></a>

-   URI格式

    DELETE /v2/\{project\_id\}/os-keypairs/\{keypair\_name\}
