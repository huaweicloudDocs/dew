# 查询凭据实例<a name="ListResourceInstances"></a>

## 功能介绍

-   功能介绍：查询凭据实例。通过标签过滤，查询指定用户凭据的详细信息。


## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CSMS&api=ListResourceInstances)中调试该接口。

## URI

POST /\{version\_id\}/\{project\_id\}/csms/\{resource\_instances\}/action
