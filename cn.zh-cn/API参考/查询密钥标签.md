# 查询密钥标签<a name="dew_02_0043"></a>

## 功能介绍<a name="section1176242255117"></a>

标签管理服务需要使用该接口查询指定密钥的全部标签数据。

## 示例<a name="section13892161018553"></a>

如下以查询密钥标签为例。

-   请求样例

    无

-   响应样例

    ```
    {         "tags": [
               {
                "key": "key1",
                "value": "value1"
               },
               {
                "key": "key2",
                "value": "value3"
               }
               ],
               "existTagsNum":2 
    }
    ```

    或

    ```
    {     
            "error": {        
                  "error_code": "KMS.XXXX",         
                  "error_msg": "XXX"    
              } 
     }
    ```
