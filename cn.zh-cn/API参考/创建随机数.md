# 创建随机数<a name="CreateRandom"></a>

## 功能介绍

生成8\~8192bit范围内的随机数。 生成512bit的随机数。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=KMS&api=CreateRandom)中调试该接口。

## URI

POST /v1.0/\{project\_id\}/kms/gen-random

## 请求示例

```
{
  "random_data_length" : "512"
}
```

## 响应示例

**状态码为 200 时: **

请求已成功。

```
{
  "random_data" : "5791C223E87120BE4B98D168F47A58BB2A88834EEADC"
}
```

**状态码为 400 时: **

请求参数有误。

```
{
  "error" : {
    "error_code" : "KMS.XXX",
    "error_msg" : "XXX"
  }
}
```

**状态码为 403 时: **

认证失败。

```
{
  "error" : {
    "error_code" : "KMS.XXX",
    "error_msg" : "XXX"
  }
}
```
