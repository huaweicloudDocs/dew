# 验证签名<a name="ValidateSignature"></a>

## 功能介绍

-   功能介绍：使用非对称密钥的公钥对消息或消息摘要进行签名验证。

## 接口约束

-   仅支持key\_usage为SIGN\_VERIFY的非对称密钥进行验签操作。

-   使用SM2密钥签名时，仅支持对消息摘要签名。根据GBT32918国家标准，计算SM2签名值时，消息摘要不是对原始消息直接计算SM3摘要，而是对Z\(A\)和M的拼接值计算的摘要，其中M是待签名的原始消息，Z\(A\)是GBT32918中定义的用户A的杂凑值。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://console.huaweicloud.com/apiexplorer/#/openapi/KMS/doc?api=ValidateSignature)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。
