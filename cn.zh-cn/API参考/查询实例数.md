# 查询实例数<a name="ShowUserInstances"></a>

## 功能介绍

-   功能介绍：查询实例数，获取用户已经创建的用户主密钥数量。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://console.huaweicloud.com/apiexplorer/#/openapi/KMS/doc?api=ShowUserInstances)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。
