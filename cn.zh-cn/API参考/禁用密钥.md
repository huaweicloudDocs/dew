# 禁用密钥<a name="DisableKey"></a>

## 功能介绍

禁用密钥，密钥禁用后不可以使用。

## 接口约束

\["密钥为启用状态才能禁用密钥。"\]

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=KMS&api=DisableKey)中调试该接口。

## URI

POST /v1.0/\{project\_id\}/kms/disable-key

```
{
  "key_id" : "0d0466b0-e727-4d9c-b35d-f84bb474a37f"
}
```

**状态码为 200 时: **

请求已成功。

```
{
  "key_info" : {
    "key_id" : "bb6a3d22-dc93-47ac-b5bd-88df7ad35f1e",
    "key_state" : "3"
  }
}
```

```
{
  "error" : {
    "error_code" : "KMS.XXX",
    "error_msg" : "XXX"
  }
}
```
