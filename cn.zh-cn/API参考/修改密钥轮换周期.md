# 修改密钥轮换周期<a name="UpdateKeyRotationInterval"></a>

## 功能介绍

修改用户主密钥轮换周期。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=KMS&api=UpdateKeyRotationInterval)中调试该接口。

## URI

POST /v1.0/\{project\_id\}/kms/update-key-rotation-interval
