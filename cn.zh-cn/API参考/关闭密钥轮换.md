## 响应消息<a name="sfadd53a5f4714e8f87811818d62d0296"></a>

无

## 示例<a name="section9910129105318"></a>

如下以关闭密钥ID为“0d0466b0-e727-4d9c-b35d-f84bb474a37f“密钥的密钥轮换为例。

-   请求样例

    ```
    {
        "key_id": "0d0466b0-e727-4d9c-b35d-f84bb474a37f"
    }
    ```

-   响应样例

    ```
    {
    }
    ```

    或

    ```
    {
        "error": {
            "error_code": "KMS.XXXX",
            "error_msg": "XXX"
        }
    }
    ```
