# 查询凭据的版本状态<a name="ShowSecretStage"></a>

## 功能介绍

查询指定凭据版本状态标记的版本信息。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://console.huaweicloud.com/apiexplorer/#/openapi/CSMS/doc?api=ShowSecretStage)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI<a name="atuogenerate_2"></a>

GET /v1/\{project\_id\}/secrets/\{secret\_name\}/stages/\{stage\_name\}
