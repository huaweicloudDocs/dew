# 查询凭据的版本状态<a name="ListSecretStage"></a>

## 功能介绍

查询指定凭据版本状态标记的版本信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CSMS&api=ListSecretStage)中调试该接口。

## URI

GET /v1/\{project\_id\}/secrets/\{secret\_name\}/stages/\{stage\_name\}
