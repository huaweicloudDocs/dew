## 示例<a name="section1776515304561"></a>

-   请求样例

    ```
    {
        "key_id": "0d0466b0-e727-4d9c-b35d-f84bb474a37f"
    }
    ```

-   响应样例

    ```
    {
        "key_rotation_enabled"：true,
        "rotation_interval"：30,
        "last_rotation_time"："1501578672000",
        "number_of_rotations"：3
    }
    ```

    或

    ```
    {
        "error": {
            "error_code": "KMS.XXXX",
            "error_msg": "XXX"
        }
    }
    ```
