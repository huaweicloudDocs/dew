# 查询失败的任务信息<a name="ListFailedTask"></a>

## 功能介绍

查询绑定、解绑等操作失败的任务信息。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://console.huaweicloud.com/apiexplorer/#/openapi/KPS/doc?api=ListFailedTask)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI<a name="atuogenerate_2"></a>
