## 响应消息<a name="section85549592421"></a>

无

## 示例<a name="section166631211114317"></a>

如下以添加标签键为“DEV“，标签值为“DEV1“为例。

-   请求样例

    ```
    {   
      "tag":
         {  
            "key":"DEV",
            "value":"DEV1"
         }
    }
    ```

-   响应样例

    ```
    { 
    }
    ```

    或

    ```
    {    
           "error": {        
                  "error_code": "KMS.XXXX",        
                  "error_msg": "XXX"     
         } 
    }
    ```
