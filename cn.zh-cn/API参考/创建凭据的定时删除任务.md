# 创建凭据的定时删除任务<a name="DeleteSecretForSchedule"></a>

## 功能介绍

指定延迟删除时间，创建删除凭据的定时任务，可设置7\~30天的的延迟删除时间。

## 接口约束

处于“定时删除”状态的凭据对象无法更新元数据信息，也无法查看凭据值。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CSMS&api=DeleteSecretForSchedule)中调试该接口。

## URI

POST /v1/\{project\_id\}/secrets/\{secret\_name\}/scheduled-deleted-tasks/create
