# 查询SSH密钥对列表<a name="ListKeypairs"></a>

## 功能介绍

查询SSH密钥对列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=KPS&api=ListKeypairs)中调试该接口。

## URI

GET /v3/\{project\_id\}/keypairs

## 请求示例

无

## 响应示例

**状态码： 200**

请求已成功。

```
{
  "keypairs" : [ {
    "keypair" : {
      "name" : "1hprr3TI",
      "type" : "ssh",
      "scope" : "user",
      "public_key" : "ssh-rsa AAAAB3NzaC1yc2EAAAADAQABjV8GvwpSs.....",
      "fingerprint" : "65:ca:87:0a:16:86:59:ea:57:ea:18:37:58:e2:04:b0",
      "is_key_protection" : false,
      "frozen_state" : 0
    }
  } ]
}
```

**状态码： 400**

Error response。

```
{
  "error_code" : "KPS.XXX",
  "error_msg" : "XXX"
}
```
