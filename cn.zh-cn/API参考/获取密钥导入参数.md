# 获取密钥导入参数<a name="dew_02_0035"></a>

## 功能介绍<a name="section2809137816511"></a>

获取导入密钥的必要参数，包括密钥导入令牌和密钥加密公钥。

>![](public_sys-resources/icon-note.gif) **说明：**   
>返回的公钥类型默认为RSA\_2048。
