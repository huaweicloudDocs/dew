# 获取密钥导入参数<a name="CreateParametersForImport"></a>

## 功能介绍

获取导入密钥的必要参数，包括密钥导入令牌和密钥加密公钥。

## 接口约束

返回的公钥类型默认为RSA\_2048。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=KMS&api=CreateParametersForImport)中调试该接口。

## URI

POST /v1.0/\{project\_id\}/kms/get-parameters-for-import

## 请求示例

```
{
  "key_id" : "0d0466b0-e727-4d9c-b35d-f84bb474a37f",
  "wrapping_algorithm" : "RSAES_OAEP_SHA_1"
}
```
