# 查询版本信息列表<a name="ShowVersions"></a>

## 功能介绍

-   功能介绍：查询API版本信息列表。

您可以在[API Explorer](https://console.huaweicloud.com/apiexplorer/#/openapi/KMS/doc?api=ShowVersions)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。
