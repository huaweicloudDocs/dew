# 查询公钥信息<a name="GetPublicKey"></a>

## 功能介绍

查询用户指定非对称密钥的公钥信息。

## 接口约束

对称密钥无法使用该操作进行查询。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=KMS&api=GetPublicKey)中调试该接口。

## URI

POST /v1.0/\{project\_id\}/kms/get-publickey
